Create a New Branch
When you create a new branch, a new pointer is created pointing to the same commit you're currently on.
git branch testing
2
Understanding HEAD
HEAD is a special pointer that tells Git which branch you're currently on. Creating a branch does NOT switch HEAD to the new branch.
cat .git/HEAD
// HOW GIT TRACKS BRANCHES
  • HEAD is a pointer to the current branch reference
  • Creating a branch does NOT switch HEAD automatically
  • Branch pointers are stored in .git/refs/heads/
  • Each branch file contains the SHA1 of the latest commit
# View current HEAD
$ cat .git/HEAD
ref: refs/heads/master

# View branch pointer
$ cat .git/refs/heads/master
f30ab... (SHA1 hash)
